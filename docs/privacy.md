# Privacy

Live policy URL: `https://quotecraft-ai-zeta.vercel.app/privacy`

QuoteCraft AI processes project details entered by the user, such as service type, project size, location, tier, urgency, and extras, in order to generate deterministic service-business quotes.

- Quote generation is based on request inputs passed to MCP tools at runtime.
- Pricing logic is config-driven from local JSON files bundled with the deployment.
- No hidden pricing enrichment or third-party quote APIs are used for pricing output.
- Hosting and operational request handling may involve OpenAI and Vercel infrastructure.
- Users should avoid submitting highly sensitive personal or regulated data.

The public submission-facing version of this policy is served from the live `/privacy` route.
