# QuoteCraft AI Brand Assets

Generated app icons:

- `quotecraft-icon-light.svg`
- `quotecraft-icon-dark.svg`
- `quotecraft-icon-industrial-light.svg`
- `quotecraft-icon-industrial-dark.svg`

Suggested usage for ChatGPT app submission:

- Export each SVG to a square PNG at 1024x1024.
- Upload the light version in the light slot.
- Upload the dark version in the dark mode slot.

If you want a more contractor / industrial look, use:

- `quotecraft-icon-industrial-light.svg`
- `quotecraft-icon-industrial-dark.svg`

The icon is designed to survive circular cropping by keeping the main mark centered and away from the outer edges.
